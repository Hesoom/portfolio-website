
# My Portfolio Website

A simple portfolio site built with Flask + HTML/CSS.

## How to run
```
python -m venv .venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install -r requirements.txt
python app.py
```
Visit http://127.0.0.1:5000/
